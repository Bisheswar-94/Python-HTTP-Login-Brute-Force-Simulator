# Python HTTP Login Brute Force Simulator

## Overview

Python HTTP Login Brute Force Simulator is a beginner cybersecurity project that demonstrates how authentication attacks work against web login forms.

This project sends HTTP POST requests with different username and password combinations to simulate brute force login attempts.

---

## Features

* HTTP POST login testing
* Username and password wordlists
* Response validation
* Delay between requests
* Logging system
* Save valid credentials
* Clean terminal output

---

## Technologies Used

* Python 3
* Requests Library

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Bisheswar-94/Python-HTTP-Login-Brute-Force-Simulator.git
```

Move into the project folder:

```bash
cd Python-HTTP-Login-Brute-Force-Simulator
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

Run the script:

```bash
python brute_force.py
```
